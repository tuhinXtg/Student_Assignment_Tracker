# System Design

## Architecture

Frontend → Backend API → Database

## Frontend
- React
- TypeScript
- Vite

## Backend
- FastAPI

## Database
- MongoDB (NoSQL)

## Main Components

### User Management
- Login
- Registration

### Assignment Management
- Create Assignment
- Update Assignment
- Delete Assignment

### Course Management
- Add Course
- View Course Assignments

## System Flow

1. User logs in.
2. User creates assignments.
3. Data is sent to backend API.
4. Backend stores data in MongoDB.
5. Frontend displays updated information.
