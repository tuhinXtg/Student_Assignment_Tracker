# Functional Requirements

## FR-01 User Registration

The system shall allow users to create an account using:

* Name
* Email Address
* Password

## FR-02 User Authentication

The system shall authenticate registered users using email and password.

## FR-03 User Logout

The system shall allow users to securely log out of the application.

## FR-04 Create Assignment

The system shall allow users to create assignments with:

* Title
* Description
* Course Name
* Due Date
* Priority Level

## FR-05 View Assignments

The system shall display all assignments associated with the logged-in user.

## FR-06 Update Assignment

The system shall allow users to modify assignment details.

## FR-07 Delete Assignment

The system shall allow users to delete assignments.

## FR-08 Assignment Status Tracking

The system shall support the following statuses:

* Pending
* In Progress
* Completed

## FR-09 Dashboard

The system shall provide a dashboard displaying:

* Total Assignments
* Pending Assignments
* Completed Assignments
* Upcoming Deadlines

## FR-10 Search and Filter

The system shall allow users to:

* Search assignments
* Filter by course
* Filter by status
* Sort by due date

## FR-11 Notifications

The system shall notify users about upcoming deadlines.

## FR-12 Data Persistence

The system shall store all assignment data in MongoDB.

## FR-13 User Profile Management

The system shall allow users to update profile information.

## FR-14 Responsive Interface

The system shall provide functionality on desktop, tablet, and mobile devices.
