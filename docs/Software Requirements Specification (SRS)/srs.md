# Software Requirements Specification (SRS)

## Project Overview

Student Assignment Tracker is a web application that helps students organize, manage, and track their assignments.

## Users

- Students
- Admin (optional)

## Technologies

- Frontend: Vite + TypeScript
- Backend: FastAPI / Spring Boot
- Database: NoSQL

## Core Features

### User Management
- User registration
- User login/logout
- User profile management

### Assignment Management
- Add new assignments
- Update assignment details
- Delete assignments
- Mark assignments as completed

### Assignment Tracking
- View pending assignments
- View completed assignments
- Track deadlines
- Search and filter assignments

## Functional Requirements

- Users can create and manage assignments
- System stores assignment information
- System displays assignment status
- Users can monitor task progress

## Non-Functional Requirements

- Simple and user-friendly interface
- Fast response time
- Secure user data
- Reliable system performance

## Future Improvements

- Assignment reminders
- Calendar integration
- Mobile app support

## Summary

The system provides an easy way for students to manage assignments and improve productivity.
