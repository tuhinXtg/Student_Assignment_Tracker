# Problem Statement

## Background

Students often manage multiple assignments from different courses simultaneously. Tracking deadlines manually through notebooks, calendars, or memory can be inefficient and prone to errors.

## Current Problems

* Missing assignment deadlines.
* Difficulty organizing assignments from multiple courses.
* Lack of centralized tracking.
* Poor visibility of assignment progress.
* Inefficient time management.

## Proposed Solution

Develop a Student Assignment Tracker that allows students to manage assignments, monitor deadlines, and track completion status through a user-friendly web application.

## Expected Outcomes

* Reduced missed deadlines.
* Improved organization of academic tasks.
* Better productivity and planning.
* Increased student satisfaction.
