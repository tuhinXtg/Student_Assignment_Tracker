# Surveys

## Objective
To collect quantitative data regarding assignment management habits among students.

## Survey Sample Size
50 Students

## Survey Questions

1. Do you currently use a tool to track assignments?
2. Have you missed an assignment deadline in the last semester?
3. How many assignments do you manage per semester?
4. Would you use an Assignment Tracker application?
5. Which features are most important?

## Survey Results

### Current Tracking Methods
- Notebook: 30%
- Mobile Notes: 25%
- Calendar Apps: 20%
- No Tracking Method: 25%

### Missed Deadlines
- Yes: 60%
- No: 40%

### Interest in Application
- Interested: 88%
- Not Interested: 12%

### Most Requested Features
1. Deadline reminders
2. Dashboard view
3. Assignment status tracking
4. Course categorization

## Conclusion
Survey results indicate a strong demand for a centralized assignment management system.
