# Information Gathering

## Objective
To understand the challenges students face in managing academic assignments and deadlines.

## Methods Used
1. Interviews
2. Surveys
3. Observation of student workflow
4. Literature review of productivity applications

## Information Collected

### Assignment Management
- Number of assignments per semester
- Assignment complexity
- Submission methods

### Deadline Tracking
- Methods currently used
- Frequency of missed deadlines

### User Preferences
- Preferred notification methods
- Desired application features

## Findings
- Most students use multiple tools to track assignments.
- Many students occasionally miss deadlines.
- Students prefer a single platform for assignment management.
- Reminder notifications are highly desired.
