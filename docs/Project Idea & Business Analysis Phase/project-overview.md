# Project Overview

## Project Title

Student Assignment Tracker

## Project Description

Student Assignment Tracker is a web-based application designed to help students organize, monitor, and manage their academic assignments efficiently. The system enables students to record assignments, track deadlines, monitor submission status, and receive reminders about upcoming due dates.

## Project Objectives

* Help students manage assignments effectively.
* Reduce missed deadlines.
* Improve academic productivity.
* Provide a centralized platform for assignment tracking.

## Target Users

* University students
* College students
* Academic institutions

## Technology Stack

### Frontend

* React
* TypeScript
* Vite

### Backend

* FastAPI

### Database

* MongoDB (NoSQL)

## Key Features

* User registration and login
* Assignment creation and management
* Deadline tracking
* Assignment status updates
* Reminder notifications
* Dashboard overview

## Expected Benefits

* Better time management
* Increased assignment completion rate
* Improved academic performance
