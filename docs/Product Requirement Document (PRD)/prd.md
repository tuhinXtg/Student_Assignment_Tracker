# Product Requirements Document (PRD)

## Product Name

Student Assignment Tracker

## Product Vision

Enable students to efficiently manage assignments, deadlines, and academic tasks through a centralized web application.

## Problem Statement

Students often struggle to organize assignments from multiple courses, resulting in missed deadlines and poor time management.

## Goals

### Business Goals

* Improve student productivity.
* Reduce missed assignment submissions.
* Provide a simple academic task management solution.

### User Goals

* Track assignments easily.
* Receive deadline reminders.
* View assignment progress.
* Organize assignments by course.

## Target Audience

* University Students
* College Students

## Functional Requirements

### User Management

* User Registration
* User Login
* User Logout

### Assignment Management

* Create Assignment
* Edit Assignment
* Delete Assignment
* View Assignment Details

### Assignment Tracking

* Set Due Date
* Update Status

  * Pending
  * In Progress
  * Completed

### Dashboard

* Upcoming Deadlines
* Assignment Statistics
* Recent Activities

### Notification System

* Due Date Reminders
* Upcoming Deadline Alerts

## Non-Functional Requirements

### Performance

* Page load time under 3 seconds.

### Security

* Password hashing.
* Secure authentication.

### Usability

* Responsive design.
* Easy navigation.

### Reliability

* 99% system availability.

## Technology Stack

Frontend:

* React
* TypeScript
* Vite

Backend:

* FastAPI

Database:

* MongoDB

## Success Metrics

* Number of active users.
* Percentage of assignments completed.
* Reduction in missed deadlines.
* User satisfaction ratings.
