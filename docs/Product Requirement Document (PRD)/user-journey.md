# User Journey

## Scenario
A student wants to track a newly assigned coursework deadline.

### Stage 1: Registration

User Actions:
- Opens website.
- Creates account.

User Goal:
- Gain access to the platform.

System Response:
- Account created successfully.

---

### Stage 2: Login

User Actions:
- Enters email and password.

User Goal:
- Access dashboard.

System Response:
- Dashboard displayed.

---

### Stage 3: Add Assignment

User Actions:
- Clicks "Add Assignment".
- Enters assignment details.
- Sets due date.

User Goal:
- Save assignment.

System Response:
- Assignment added successfully.

---

### Stage 4: Track Progress

User Actions:
- Views assignment list.
- Updates status.

User Goal:
- Monitor progress.

System Response:
- Status updated.

---

### Stage 5: Receive Reminder

User Actions:
- Receives notification before deadline.

User Goal:
- Complete assignment on time.

System Response:
- Reminder displayed.

---

### Stage 6: Complete Assignment

User Actions:
- Marks assignment as completed.

User Goal:
- Track completion.

System Response:
- Assignment moved to completed category.
