# User Stories

## Authentication

US-01
As a student,
I want to register an account,
So that I can use the application.

US-02
As a student,
I want to log into my account,
So that I can access my assignments.

---

## Assignment Management

US-03
As a student,
I want to create assignments,
So that I can keep track of academic work.

US-04
As a student,
I want to edit assignments,
So that I can update assignment information.

US-05
As a student,
I want to delete assignments,
So that I can remove unnecessary records.

---

## Assignment Tracking

US-06
As a student,
I want to update assignment status,
So that I can track my progress.

US-07
As a student,
I want to view upcoming deadlines,
So that I can prioritize my work.

---

## Notifications

US-08
As a student,
I want to receive reminders,
So that I do not miss deadlines.

---

## Dashboard

US-09
As a student,
I want to view assignment statistics,
So that I can monitor my academic progress.
